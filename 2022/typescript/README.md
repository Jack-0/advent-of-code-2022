# Advent of code 2022

Typescript and node

# Running the solutions

`yarn start -d 1 -e`

# Running the tests

All tests
`yarn test`

Specific day
`yarn test -r day01`

# options

Select a day
`-d <day>`

Optional use example data
`-e`

# Debug support

brave://inspect/#devices
`yarn node --inspect-brk --require ts-node/register src/index.ts -d 0 -e`
